Instalar o Git

Verificando o Git


    Para verificar se você já tem o Git instalado, abra seu aplicativo de terminal.
  • Se você estiver em um Mac, procure por um aplicativo de prompt de comando chamado "Terminal".
  • Se você estiver em uma máquina Windows, abra o prompt de comando do Windows ou o "Git Bash".

    • Após abrir o aplicativo de terminal, digite git version. A saída informará qual versão do Git está instalada ou alertará que git se trata de um comando desconhecido. Se for um comando desconhecido, continue lendo e descubra como instalar o Git.

Exemplo:

$ git version
bash: git: command not found

Instalar o Git usando o GitHub Desktop

A instalação do GitHub Desktop também instalará a versão mais recente do Git, caso você ainda não a tenha. Com o GitHub Desktop, você obtém uma versão de linha de comando do Git com uma interface gráfica robusta. Independentemente de você ter o Git instalado ou não, o GitHub Desktop oferece uma ferramenta de colaboração simples para o Git. GitHub Desktop Aqui

Baixar Git no Windows


  1. Pesquise o instalador mais recente do git para seu Windows e instale-o.
  2. Após Instalar abra o instalador e siga o passo a passo para concluir a instalação.
  3. Abra o prompt de comando do Windows (ou Git Bash se você optou por não usar o prompt de comando padrão do Git do Windows durante a instalação do Git).
  4. Digite o comando git version para verificar se o git foi instalado.

    5. Exemplo:

    $ git version
    git version 2.49.0.windows.1

Baixar Git no Mac


  1. Pesquise o instalador mais recente do git para seu MacOs e instale-o.
  2. Após Instalar abra o instalador e siga o passo a passo para concluir a instalação.
  3. Abra o prompt de comando "terminal" e digite git version para verificar se o Git foi instalado.

    4. Exemplo:

    $ git --version
    git version 2.44.0

Instalar o Git no Debian/Ubuntu (Linux)


  1. Pacotes Git estão disponíveis usando apt.
  2. É uma boa ideia garantir que você esteja executando a versão mais recente. Para isso, acesse o prompt de comando e execute o seguinte comando para garantir que tudo esteja atualizado: sudo apt-get update.
  3. Para instalar o Git, execute o seguinte comando: sudo apt-get install git-all.
  4. Depois que a saída do comando for concluída, você pode verificar a instalação digitando: git version
