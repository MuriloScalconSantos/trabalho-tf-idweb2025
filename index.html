Guia Básico do GitHub

Criar uma Conta no GitHub

  1. Acesse github.com
  2. Clique em "Sign up"
  3. Preencha seus dados (e-mail, senha, nome de usuário)
  4. Verifique seu e-mail

Criar um Repositório

  1. Após logar, clique no ícone "+" no canto superior direito
  2. Selecione "New repository"
  3. Dê um nome ao repositório
  4. Escolha se será público ou privado
  5. Clique em "Create repository"

Realizar um Fork

  1. Navegue até o repositório que deseja copiar
  2. Clique no botão "Fork" no canto superior direito
  3. Selecione sua conta como destino
  4. Aguarde a cópia ser criada

Realizar um Pull Request

  1. No seu repositório forkado, clique em "Pull requests"
  2. Clique em "New pull request"
  3. Selecione os branches para comparação
  4. Revise as alterações
  5. Clique em "Create pull request"
  6. Adicione um título e descrição
  7. Clique em "Create pull request" novamente
